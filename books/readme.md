## books


1. Feeling Good: The New Mood Therapy — David D. Burns

This book explains how negative thoughts affect emotions and behavior using Cognitive Behavioral Therapy (CBT). It teaches techniques for identifying distorted thinking patterns such as overthinking, self-criticism, and hopelessness, then replacing them with healthier and more balanced thoughts.
It is very useful for building AI systems that analyze user emotions, detect negative thinking patterns, and generate supportive therapeutic responses.

2. On Becoming a Person — Carl Rogers

This book focuses on human-centered psychotherapy and empathetic communication. It teaches how therapists listen, understand emotions deeply, and create safe, non-judgmental conversations.
It is highly useful for improving the conversational style of an AI therapy assistant, especially empathy, emotional understanding, reflective listening, and supportive dialogue generation.

3. Cognitive Behavior Therapy: Basics and Beyond — Judith Beck

This is one of the most practical and professional CBT books. It explains how real therapy sessions are structured, how therapists guide conversations, identify cognitive distortions, and help patients change harmful thought patterns.
It is extremely useful for designing the therapy logic, dialogue flow, memory system, and reasoning process of an AI therapeutic assistant.

4. Cognitive Behavioural Therapy (CBT) Skills Workbook — NHS

This workbook contains practical exercises, worksheets, emotional tracking tools, and self-help techniques for anxiety, depression, stress, and panic.
It is useful for adding interactive features to an AI therapy platform such as mood tracking, journaling, emotional check-ins, coping exercises, breathing exercises, and guided self-reflection activities.
